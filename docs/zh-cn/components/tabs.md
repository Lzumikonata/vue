<h3>zi-tabs</h3>

| 属性 | 描述 | 类型 | 推荐值 | 默认
| ---------- | ---------- | ---- |  -------------- | ------ |
| **label-selected**(event) | 开关点击事件 | `event` | - | - |

<br/>

<h3>zi-tabs-item</h3>

| 属性 | 描述 | 类型 | 推荐值 | 默认
| ---------- | ---------- | ---- |  -------------- | ------ |
| **label**(必须属性) | 文本 | `string` | - | - |
| **value**(必须属性) | 值，唯一性 | `string` | - | - |
| **active** | 当前激活状态 | `boolean` | - | `false` |
